マップチップの選定と画像作成ツール

概要

/ORIGINAL_ASSETS/map配下の画像は、FirstSeedMaterialで配布されていた、RPGツクール想定のマップチップ各種。 自作ゲームで利用するには規模が大きすぎるので、自分ゲーム向けに選定して、1枚画像を作成したい。 このプロジェクトでは、そのためのツールを作成する。

※CSSのattr()に対応していないブラウザでは正しく動作しない

AIに相談

AIへの入力メモ

実現したい操作：

  1. 複数の元画像a,b,c,...を元に、1枚の結果画像を作成したい
  2. 元画像から任意の矩形を選択する（a1:x1,y1-x2,y2など）
  3. 矩形a1,a2,..を結果画像の任意の位置に配置する

質問：

  1. 上記の操作を行なうシェルスクリプトが作りたい場合、ImageMagickを使うのが一般的でしょうか？
  2. ImageMagickよりも良い選択（分かりやすさ、環境構築のしやすさ、結果コードの短さなどの観）があれば提案して欲しい
  3. サンプルコードを提示して欲しい

AI(Copilot)の回答

以下は、複数の画像から矩形領域を切り出し、1枚のキャンバスに合成する例です。

#!/bin/bash

# 元画像と切り出し座標の定義
declare -A crops
crops[a.jpg]="100x100+10+10"
crops[b.jpg]="80x120+30+40"
crops[c.jpg]="50x50+0+0"

# 結果画像のサイズ
canvas_width=300
canvas_height=200
output="result.png"

# 空のキャンバス作成（白背景）
convert -size ${canvas_width}x${canvas_height} xc:white "$output"

# 合成位置の定義（画像ごとに任意に設定）
positions=("0x0" "100x50" "200x100")
i=0

# 各画像を切り出して合成
for img in "${!crops[@]}"; do
  crop="${crops[$img]}"
  pos="${positions[$i]}"
  tmp="tmp_$i.png"

  # 矩形切り出し
  convert "$img" -crop "$crop" +repage "$tmp"

  # 合成
  composite -geometry +${pos/x/+}+"${pos#*x}" "$tmp" "$output" "$output"

  ((i++))
done

# 一時ファイル削除
rm tmp_*.png

📝 補足

  • -crop WxH+X+Y：画像から矩形領域を切り出す
  • composite -geometry +X+Y：指定位置に画像を合成
  • xc:white：白背景のキャンバスを作成
  • +repage：切り出し後の余白を除去
ImageMagickコマンド調査
  • Command-line Tools: Convert | ImageMagick

空のキャンバス（画像）作成（透過背景）

背景が透明ならxc:noneにします

convert -size "${WIDTH}x${HEIGHT}" xc:none "${CANVASFILE}"
  • -size - Command-line Options | ImageMagick

-size width[xheight][+offset] Set the width and height of the image.

矩形の切り出し

  • -crop - Command-line Options | ImageMagick
  • Image Geometry - Command-line Processing | ImageMagick

-crop geometry{@}{!} Cut out one or more rectangular regions of the image.

サイズ 一般的な説明（実際の動作はオプションや設定によって異なる場合があります）
幅×高さ 高さと幅の最大値が指定され、アスペクト比が維持されます。 
# crop ex: -crop '100x200+20+40'
convert "${SRCFILE}" -crop "${WIDTH}x${HEIGHT}+${X}+${Y}" +repage "${CHIPFILE}"

画像の合成

切り出した矩形を空のキャンバスに合成する。

  • Command-line Tools: Composite | ImageMagick
composite -geometry +"${X}"+"${Y}" "${CHIPFILE}" "${CANVASFILE}" "${OUTPUTFILE}"

※複数の${CHIPFILE}を${CANVASFILE}に重ねたい場合は、${OUTPUTFILE}に${CANVASFILE}を指定して上書きする。

操作方法

  1. 📜Commandボタン押下でコマンドダイアログを表示する。
  2. ダイアログ内の📔Copyボタンでコマンドをクリップボードにコピーする
  3. プロジェクトの階層（prj/04.mapchip_selector）でコマンド実行（ペースト）
    • ※ImageMagicが必要
  4. mapchip.pngが作成される
    • ※prj/assets/image/mapへのコピーは自分でやる
