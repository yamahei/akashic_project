# フォント画像の使い方
フォント画像を g.Bitmapfont で利用するとき、それぞれの文字の大きさや画像上の位置といった情報(グリフエリア)が必要になります。グリフエリアを保存したファイル（グリフエリアファイル）を用意してありますので、各フォント画像ファイルに対応したグリフエリアファイルをご使用ください。


| フォント画像ファイル | グリフエリアファイル | サイズ |
|:---------------------|:---------------------|:-------|
| font16_1.png         | glyph_area_16.json   | 16x16  |
| font16_2.png         | glyph_area_16.json   | 16x16  |
| font20_1.png         | glyph_area_20.json   | 20x20  |
| font20_2.png         | glyph_area_20.json   | 20x20  |
